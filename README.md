# 💣 弾着タイマー (Impact Timer)

サバゲーやシミュレーションゲームでの作戦進行・弾着観測に特化した、PWA（プログレッシブウェブアプリ）対応のマルチタイマーアプリです。
複数の分隊、複数の弾着タイミングを同時にかつ直感的に管理できます。

## ✨ 主な機能
* **マルチタイマー管理**: 最大5個分隊、各分隊最大50発までのタイマーを一括生成。
* **個別秒数設定**: 分隊ごとに作戦行動（タイマー秒数）を個別に割り当て可能。
* **直感的なタップ操作**: タップで即スタート、ダブルタップで即リセット。
* **視認性の高いUI**: ダークモード完全対応。残り5秒で赤色点滅、終了時に赤色点灯で視覚的に通知。
* **オートフォーカス**: タイマー終了時、次に待機しているタイマーへ自動でスクロール。
* **作戦時計（経過時間）**: 最初のタイマーを起動した瞬間から、全体の作戦経過時間を自動計測。
* **オフライン動作**: ホーム画面に追加すれば、電波の悪いフィールドでも爆速起動・オフライン動作が可能。

---

## 📱 アプリのインストール方法（ホーム画面に追加）
本アプリはブラウザから直接スマホのホーム画面に追加し、ネイティブアプリのように全画面で使用することを推奨しています。

* **iPhone (Safari) の場合**
  1. ブラウザでアプリのURLを開く
  2. 画面下部の「共有ボタン（四角から矢印が飛び出したアイコン）」をタップ
  3. 「ホーム画面に追加」をタップ
* **Android (Chrome) の場合**
  1. ブラウザでアプリのURLを開く
  2. 画面右上（または下部）の「メニュー（︙）」をタップ
  3. 「ホーム画面に追加」をタップ

## 📖 使い方

### 1. 初期設定画面
アプリを開くと、まず作戦に合わせた設定画面が表示されます。
* **一括セットする秒数**: 基本となるタイマーの秒数を設定します。
* **分隊ごとに秒数を個別設定**: ここを「オン」にすると、分隊ごとに異なる秒数（第1分隊は30秒、第2分隊は60秒など）を設定できるようになります。
* **分隊の数**: 管理する分隊数（1〜5）を選択します。
* **各分隊の弾数**: 各分隊が何発撃つか（タイマーをいくつ用意するか）を設定します。

設定が完了したら、一番下の **「タイマー画面へ」** をタップしてください。

### 2. タイマー画面の操作
生成されたタイマーボタンを使って時間の管理を行います。

* **▶️ スタート**
  * 待機中（灰色）のタイマーを **1回タップ** するとカウントダウンが始まります（黄色になります）。
  * 最初のタイマーをスタートさせた瞬間に、画面最上部の「経過時間（作戦時計）」も自動的に動き出します。
* **⏹️ 停止・リセット**
  * 作動中のタイマーを **ダブルタップ（素早く2回タップ）** すると、即座に停止して元の秒数にリセットされます。押し間違えた際にご活用ください。
* **🔔 終了の通知**
  * 残り5秒を切ると **赤色に点滅** して警告します。
  * 0秒になると **赤色に点灯** し、自動的に「次のタイマー」が画面中央に来るようにスクロールします。

作戦が終了したら、画面最下部の **「設定に戻る」** をタップすることで、すべてのタイマーと経過時間がリセットされ、初期画面に戻ります。
